How REQUEST A GAME

Basic info.

Following the rules of each game is important before anything else.
And of course common sense, it's a free non-profit work, sometimes it can take a while, so don't request it multiple times.


Mandatory

It is necessary to say the name of the game (Not abbreviated, without links, full name) Ex: Metal Slug X.

You have to say the platform: Steam, GOG, Epic, ETC.


Optional

Desired functions Ex: "Inf Money", "Skip timer of exit mission".

Similar design desired.

Executable or DLL or Framework.

Hotkeys (shortcut keys) ex: Shift + F1 = Active Infinity Jump.



Easy no? But no Idea? See this exemple

Game name: Bayonetta.
Plataform: Steam.
if you can make this functions
Infinity Jump
God Mode
thats all.
Thanks for you work
